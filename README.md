# TS弹层播放器

基于 TypeScript 和原生的仿小米官网的弹层播放器

## Feature

- [x] 基本播放器模样
    - [x] 全屏
    - [x] 音量调节
    - [x] 播放进度条自动变化
    - [x] 播放进度拖动
    - [ ] 点击静音按钮
    - [ ] 点击进度某一点跳转播放
    - [ ] 播发几秒后自动隐藏控件
- [x] 基于 `TypeScript` 的原生实现
- [x] 可配置的内容：
    - [x] 定位
    - [x] 大小
    - [x] 是否蒙层
    - [x] 是否自动播放
- [ ] React 组件 
- [ ] Vue3 组件
- [ ] 悬浮在视频中间的播放按钮
- [ ] 定位大小按播放器百分比计算
- [ ] 细致交互的动画
- [ ] 代码整体优化
